# 🛒 IoT Tech Store (E-commerce Website)

## 📌 Overview

This is a simple **E-commerce website** built using **HTML, CSS, and JavaScript**.
It allows users to browse IoT products, add items to cart, and navigate through different sections like Home, Products, Cart, Login, and Contact.

---

## 🚀 Features

* 🏠 Home page with hero section
* 🛍️ Product listing (Arduino, Raspberry Pi, etc.)
* 🛒 Add to Cart functionality
* 🔐 Login UI (basic frontend)
* 📱 Responsive design
* 🔄 Single-page navigation using JavaScript

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```
E-commerce/
│── index.html
│── style.css (optional if separated)
│── script.js (optional if separated)
│── images/
```

---

## ⚙️ How to Run Locally

1. Download or clone the repository
2. Open the folder
3. Run `index.html` in your browser
